# Weekly Review Dashboard

A personal deployment of Tiago Forte's Weekly Review checklist, hosted on GitHub Pages for my own convenience.

🔗 **[Open the App](https://fpaulpalis.github.io/weekly-review/)**

---

## What this is

This is a simple browser-based checklist that walks me through a 30-minute Weekly Review — a habit from Tiago Forte's *Building a Second Brain* system. I downloaded the original app from his newsletter and deployed it here so I can access it anytime without hunting for the file.

## The 4-Step Process

1. **Clear Email** (5–10 min)
2. **Review Calendar** (5 min)
3. **File Notes & Clear Desktop** (5–10 min)
4. **Choose Tasks** (10 min)
